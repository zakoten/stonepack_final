# Tramonto Stone Pack

143 missing stone blocks for Minecraft **1.21.11** — server-side only, powered by Polymer.

---

## Where to put your textures

Place all PNG files (16×16) in:
```
src/main/resources/assets/stonepack/textures/block/
```
